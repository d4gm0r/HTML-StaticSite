Unix Shell Commands

Cheat-Sheet

By Nathan Larsen

Familiarity with unix shell commands is essential to my continued "survival" in this class. The following list is by no means comprehensive, but is composed of some of the commands I consider absolutely fundamental to the operation of the command line.

I hope one day to be able to say this:

"It's a unix system. I know this!" -That one girl from Jurassic Park

I therefore present this guide or cheat-sheet on unix commands.

The table below lists the commands, then provides a short description of what the command does.

---------UNIX SHELL COMMANDS----------

Command Example Description
cd

cd user

cd..

changes the directory to user

move back one directory in the path
ls ls list all files in current directory
mkdir mkdir example | rmdir example creates a directory named example
rmdir rmdir example removes example directory
rm rm examplefile.txt deletes specified file
cp cp examplefile exampledirectory copies specified file into specified directory
mv mv first.txt second.txt rename or move specified file
